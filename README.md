**MaternalAI** is a sophisticated, privacy-focused digital health assistant designed to support maternal wellness through data-driven clinical insights. Built with a "White Label" architecture, it provides professional medical assessment, medication tracking, and personalized wellness guidance in **English, Tamil, and Hindi**.

---

## ✨ Key Features

### 🏥 Clinical Health Assessment
* **Risk Engine:** Analyzes Hemoglobin (Hb) levels, Blood Pressure (BP), and pregnancy week to calculate a clinical risk probability.
* **AI Analysis:** Provides deep-dive clinical assessments using an anonymized high-performance LLM core.
* **Direct Report View:** View comprehensive clinical roadmaps directly within the web interface.

### 📅 Personalized Weekly Guidance
* Dynamically generated wellness tips tailored to the exact stage of pregnancy.
* Medical-backed advice synchronized with the user's current vitals.

### 💊 Advanced Medication Log
* Real-time tracking of daily dosages.
* Multilingual support for medicine names and instructions.
* Persistent session tracking with easy "Delete" management.

### 🎵 Wellness & Mental Health
* **Relaxation Music:** AI-curated soundscapes suggested based on current health data to reduce prenatal stress.
* **Doctor Chat:** An empathetic, clinical-grade chatbot for 24/7 health queries.

### 📄 Professional Export
* **Instant PDF:** Generate a professional medical report with one click.
* **High Contrast Design:** Optimized for readability with a clean, dark-mode sidebar and white-card dashboard.

## 🛠️ Tech Stack
* **Frontend:** Streamlit (Custom CSS for High-Contrast UI)
* **Brain:** MaternalAI Discovery Engine (REST API based)
* **Data Processing:** Pandas & NumPy
* **Document Generation:** FPDF
* **Cloud Deployment:** Streamlit Community Cloud
